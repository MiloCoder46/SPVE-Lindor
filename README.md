# SPVE-Lindor
This mod adds volumetric aurora to Lindor for JNSQ, the config was made with Copilot do keep in mind. Anyway, install the mod by dragging the folder located in the mod folder Gamedata/SPVE-Lindor into your KSP GameData folder. The aurora for Lindor should show up, this mod requires SpacePotato's visual enhancements and KSP to actually function.
